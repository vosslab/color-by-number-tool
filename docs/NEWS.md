# News

This page presents the short, user-facing highlights for each release.

## v26.07 - 2026-07-22

### Highlights

- Turn an image into two aligned Letter pages: a blank light-gray coloring grid and a black
  numbered reference grid.
- Start with an 86 by 60 grid, or choose another square-cell density with `-g`/`--grid`; page
  orientation follows the source image unless explicitly overridden.
- Match every cell to one of the 48 supplied Aoartix marker codes with portrait-focused color
  enhancement and no global dithering.
- Review a marker-key worksheet, colored previews, assignments, marker counts, and color-distance
  metrics before coloring the final page.
